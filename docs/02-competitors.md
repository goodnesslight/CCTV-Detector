# 02. Аналіз існуючих рішень

## 2.1. Класифікація ринку

Системи відеоспостереження з ШІ-функціями умовно поділяються на чотири сегменти:

```
                       Локальне             Хмарне
                  ┌──────────────────┬──────────────────┐
   Платне         │  Hikvision /     │  Verkada         │
   (vendor)       │  Dahua AI cams   │  Eagle Eye       │
                  ├──────────────────┼──────────────────┤
   Open-source /  │  Frigate         │  (немає —        │
   безкоштовне    │  ZoneMinder      │   суперечить ідеї)│
                  │  Blue Iris*      │                  │
                  └──────────────────┴──────────────────┘
        * Blue Iris — комерційний, але single-purchase,
          без хмари. Включений як «локальне платне».
```

Цей застосунок належить до сегмента **локальне + open-source**, безпосередня
конкуренція — Frigate та ZoneMinder.

## 2.2. Огляд конкурентів

### 2.2.1. Frigate NVR

- **URL:** https://frigate.video, https://github.com/blakeblackshear/frigate
- **Ліцензія:** MIT
- **Платформа:** Docker-контейнер на Linux, інтегрується з Home Assistant
- **AI:** Coral Edge TPU (~$60 USD прискорювач) або CPU/GPU
- **Моделі:** SSD-MobileNet з COCO (детектує 80 класів), опційно YOLO
- **Інтерфейс:** Web-UI

**Переваги:** зрілий проект (з 2020), активна спільнота, інтеграція з smart-home.

**Недоліки/обмеження:**
- Орієнтований на користувачів з Linux + Docker + Home Assistant — поріг входу високий.
- Розпізнавання облич — не штатна функція (можна додати через сторонній frigate-plus, платний).
- Web-UI — потрібен сервер у мережі, не «запустив-і-працює» застосунок.

### 2.2.2. ZoneMinder

- **URL:** https://zoneminder.com
- **Ліцензія:** GPLv2
- **Платформа:** PHP/Perl/MySQL стек на Linux
- **AI:** окремий plugin `zmeventnotification` через DeepStack/Yolo

**Переваги:** найдавніший open-source NVR (з 2002), стабільний.

**Недоліки/обмеження:**
- **Архаїчна архітектура** (LAMP-стек, написаний у 2002 році).
- ШІ — як plugin, інтеграція громіздка.
- UX/UI дуже застарів (web-інтерфейс 2010-х).

### 2.2.3. Blue Iris

- **URL:** https://blueirissoftware.com
- **Ліцензія:** комерційна, ~$70 одноразово (Pro-версія)
- **Платформа:** тільки Windows
- **AI:** інтеграція з DeepStack (зовнішній сервер)

**Переваги:** stabilна Windows-програма з активною підтримкою.

**Недоліки:**
- Платне ПЗ.
- Закритий код — не дозволяє кастомізацію.
- AI як зовнішній сервіс, який треба підняти окремо.

### 2.2.4. Hikvision/Dahua AI-камери

- **Ціна:** від $300 за камеру з ON/OFF-AI або $800+ за камеру з PIPM (people counting / face)
- **Платформа:** Прошивка камери + фірмовий VMS

**Переваги:** «з коробки» — поставив, увімкнув, працює.

**Недоліки:**
- Vendor lock-in.
- Дані можуть йти на сервери виробника (історично доведено для Hikvision у 2017).
- Закрита логіка — ні аудит, ні зміна правил.

### 2.2.5. Verkada / Eagle Eye Networks

- **Модель:** хмарна підписка
- **Ціна:** $200–$500/міс. за камеру

**Переваги:** «під ключ», масштабоване, мобільні застосунки.

**Недоліки:**
- Жорстка прив'язка до підписки.
- Усі відео йдуть у хмару вендора (юридичні ризики).
- Необґрунтоване для невеликих об'єктів.

## 2.4. Позиціонування цього проекту

```
              Простота встановлення
                       ↑
                       │
          Verkada ●    │   ● Цей проект
                       │
       Hikvision ●     │
                       │
                       │   ● Blue Iris
                       │
                       │ ● Frigate
                       │
                       │
                       │       ● ZoneMinder
                       │
                       └────────────────────→
                       Контроль над даними
```

**Ніша**, у якій існуючі рішення не покривають усіх потреб одночасно:

> **Локальний, безкоштовний, з UI рівня сучасного desktop-застосунку, без потреби
> в Docker / Home Assistant / зовнішніх сервісах, з готовою функцією розпізнавання
> облич із коробки.**

Frigate та ZoneMinder — це сервери. Blue Iris — закритий комерційний продукт.
Hikvision/Verkada — хмарні / vendor-lock. Цей проект — **класичний desktop-app
з усіма AI-функціями вбудованими, який запускається подвійним кліком на Windows**.

## 2.5. Honest disclaimers

Не претендую на конкуренцію з Frigate за зрілістю, обсягом фіч (зокрема multi-camera,
RTSP record, MQTT-інтеграція). Frigate — це робочий продукт із 4-річною історією.
Цей проект — **дипломна реалізація** з акцентом на демонстрацію застосування ШІ
у домені відеоспостереження та власну архітектуру обробки потоку.

Цінність проекту — в **архітектурній ясності** (читомий Python-код, який легко
адаптувати під свій сценарій) та в **інтеграції стеку без зовнішніх сервісів**.

Подальші напрями розширення (out of diploma scope): мульти-камерна обробка,
web-UI замість Qt, REST API для інтеграцій, hardware-acceleration через TensorRT.
